# Eating Here React App

The Eating Here App is built with React Hooks + Redux. This web application is for food lovers to explore the most unusual restaurants around the world. Users can add new restaurants as well as comment and like any restaurant. The App also provides name search functionality. EnjoyðŸ˜Š!

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Feel free to check out the backend API repo <a href="https://github.com/lena0128/eat-here-rails-api-backend">here</a>.

## Installation & Usage

In the project directory, you can run:
### `npm install`
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
